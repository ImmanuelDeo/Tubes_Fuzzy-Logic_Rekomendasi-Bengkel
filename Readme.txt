PANDUAN PENGGUNAAN PROGRAM

Cara Menjalankan Program:

Opsi A: Menggunakan VS Code atau Jupyter Notebook (Lokal):
1. Pastikan Python 3 sudah terinstal di komputer.
2. Buka folder proyek ini menggunakan VS Code atau Jupyter.
3. Buka file **`Tubes_Fuzzy.ipynb`**.
4. Pastikan file `bengkel.csv` berada di direktori yang sama.
5. Klik tombol **"Run All"** (atau pilih menu *Kernel* > *Restart & Run All*) untuk mengeksekusi seluruh sel kode dari atas ke bawah.

Opsi B: Menggunakan Google Colab (Online)
1. Buka Google Colab melalui link: https://colab.research.google.com/.
2. Upload file Tubes-FuzzyLogic-Bengkel.ipynb ke Colab.
3. Upload file bengkel.csv ke bagian Files (ikon folder di sebelah kiri layar Colab).
4. Klik Run all di menu tab bagian atas
